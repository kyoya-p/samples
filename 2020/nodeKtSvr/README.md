Web Server with Kotlin on node.js
====

Build Environment
----
``` 
git clone https://github.com/kyoya-p/samples
cd samples/nodeKtSvr
```

Build & Run
----
- build
```
./gradlew npmInstall build
```
- run
```
./gradlew run
```

or 

```
node ./build/js/packages/nodeKtSvr/kotlin/nodeKtSvr.js 
```
