
## サンプルの確認
```
git clone https://github.com/webrtc/samples
cd samples
npm i 
npm start
```

## RTCPeerConnection
```:Build and Run
npm i
npx tsc
npx http-server public -d -tls 
# http://localhost:8080/public/peer-server.html
# http://localhost:8080/public/peer-client.html
```
