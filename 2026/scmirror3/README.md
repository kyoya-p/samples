# Screen Capture Mirror

ブラウザ上で全画面をミラーリングし、ブラウザからのクリック操作とキー入力を転送するWebツール。

## 機能
- **画面ミラーリング**: `navigator.mediaDevices.getDisplayMedia` を使用してブラウザに投影。
- **リモート操作**: 
  - ブラウザ上の画面をクリックすると、マウスイベントを発行。
  - ブラウザ上でキー押下でキーイベントを発生
  - **リアルタイム情報表示**: 
    - コンテンツ左上にオーバーラップ表示。
    - ビデオ上でのマウス移動に合わせて、マウスピクセル座標を表示。

## セットアップと実行

1.  **依存関係のインストール**:
    ```bash
    npm install
    ```

2.  **サーバーの起動**:
    ```bash
    node server.js -p <port>
    ```
    - port: デフォルト3003
3.  **ブラウザでアクセス**:
    `http://localhost:3003` を開く。

5.  **サーバーの終了**:
    以下のアドレスにアクセスするとサーバー終了。
    `http://localhost:3003/close`
    **停止のためのプロセスkillは絶対禁止!!**
