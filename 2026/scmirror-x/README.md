# Screen Capture Mirror

ブラウザ上でリモートウィンドウの画面をミラーリングし、ブラウザからのクリック操作を転送するシンプルなツール。

## 機能
- **画面ミラーリング**: `navigator.mediaDevices.getDisplayMedia` を使用してウィンドウをブラウザに投影。
- **リモート操作**: 
  - ブラウザ上のビデオ画面をクリックすると、ターゲットウィンドウ（またはプライマリスクリーン）に対してマウスイベントを発行。
  - **リアルタイム情報表示**: 
    - コンテンツ左上にオーバーラップ表示。
    - ビデオ上でのマウス移動に合わせて、タイトルバーにターゲット解像度に基づいたピクセル座標を表示。
    - キャプチャ対象ウィンドウのデスクトップ上での位置（Pos）とサイズ（Size）。
- **ミニマル UI**: タイトルを排し、ビデオ上にオーバーラップしたアイコンボタン（Material Icons）からキャプチャを開始可能。

## セットアップと実行

1.  **依存関係のインストール**:
    ```bash
    npm install
    ```

2.  **サーバーの起動**:
    ```bash
    node server.js -p <port>
    ```
    - port: デフォルト3000

3.  **ブラウザでアクセス**:
    `http://localhost:3000` を開く。

4.  **ターゲット設定とキャプチャ開始**:
    - 上部のメニューから操作対象のウィンドウタイトルを選択（リフレッシュボタンで一覧更新）。
    - 右上のアイコンボタンをクリックし、ミラーリングしたいウィンドウを選択する。

5.  **サーバーの終了**:
    以下のアドレスにアクセスするとサーバーを終了できる（ブラウザのタブを閉じるだけではサーバーは停止しない）。
    `http://localhost:3000/close`

## ファイル構造
- `server.js`: WebSocket サーバーおよび静的ファイル配信。
- `index.html`: ミラーリング画面および座標送信ロジック。
- `package.json`: 依存関係定義。
