# 内容
TCG BattleSpirits のデッキシミュレータ

# ルール仕様
- ゲームフォーマット: エターナル

# 操作の例

- 初期局面:
  - リザーブコア: 2
  - トラッシュコア: 0
  - フィールド: スピリット1(シンボル:赤1,コア:1), スピリット2(シンボル:緑1,コア:1)

- 操作: スピリット(コスト2: シンボル:赤1, 軽減:赤1)召喚

- 局面
  - リザーブコア: 0
  - トラッシュコア: 1
  - フィールド: スピリット1(シンボル:赤1,コア:1), スピリット2(シンボル:緑1,コア:1), スピリット3(シンボル:赤1,コア1)



# アプリ仕様

- 初期配置
  - 定義ファイルに基づきデッキ配置
- 局面においての選択肢を列挙する列挙関数
- 選択に基づき次の局面を生成する遷移関数

- 初期状態から、下記を繰り返す:
  - 局面に応じた選択肢を列挙し、ユーザによる選択を入力(終了も選択可)
  - 入力に応じ、次の局面を生成する
