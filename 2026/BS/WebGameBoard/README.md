- ゲームボード
  - フィールド(公開領域)、非公開領域、からなる
  - 非公開エリアでは両面の情報が確認できる
  - カードが最前面
  - 背面には任意のイメージを設定できる
  - リセットボタン
    - すべてのカードを裏向きでスタックし、デッキとして右上に置く
    
- カード
  - フィールド上の任意の座標に配置できる
  - ローテーション(90度回転)状態。4回ローテーションで元の状態に戻る
  - ハンドラオーバレイ
    - ハンドラ枠はスタック状態のカードをすべて納められるサイズの正方形(カードがローテーション状態でもハンドラに収まるサイズ)
    - カードの上にはハンドラアイコンがオーバレイされる
      - ハンドラアイコンは、 ドラッグ、フリップ、ローテート
  - ドラッグ&ドロップでフィールド上を移動できる
    - ドラッグ中別カードの上に乗った場合視覚的に表現
    - 別カードの上にドロップした場合スタックする
    - スタックする場合、カードのローテーション状態はスタック先に合わせる

- スタック
  - カードが重なった状態。視覚的には 2dp ずれた重ね状態で表示
    - 一番下のカードのみ10dpずらす。クリック容易にするため。
    - スタックされたカードそれぞれのハンドラは表示されない
  - フィールド上の任意の座標に配置できる
  - スタックのハンドラアイコン
    - スタックしたカード **すべて** を囲む正方形の領域とする 
    - 機能はドラッグ、フリップ、ローテート、スタックドラッグ、スプレッドモード
    - スプレッドモードアイコンにはスタック枚数を数値表示
  - スタックをドラッグする場合
    - スタック内でクリックしドラッグする場合、クリックしたカード1枚を移動する
    - スタックハンドラ内のカード以外の部分をクリックしドラッグする場合、スタック全体を移動
  - スタックかカードをドラッグしてカードかスタックの上にドロップした場合、一つのスタックとなる
    - ドロップしたカード、スタックのローテーション状態はドロップ先のスタックに合わせる
  - スタックのローテーションはスタックされたすべてのカードに対して行われる
  - スタック内のカード1枚をドラッグし自身のスタックにドロップ可能。その場合、スタックのトップにおかれる
  - スプレッドモード(横方向にカード半分ずらして広げ選択を容易にする)
