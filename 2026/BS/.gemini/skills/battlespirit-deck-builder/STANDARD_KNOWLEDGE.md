# Battle Spirits: Standard Regulation Knowledge Base (2026-)

## 1. 概要
2026年4月より導入された、ブロックアイコンが英字（例:「A」）のカードのみを使用するレギュレーション。従来の「エターナル」とは独立したカードプールと一部異なるルールを持つ。

## 2. 対戦準備・基本ルール
*   **マリガン（引き直し）**:
    *   ゲーム開始時、1度だけ手札を全てデッキに戻して引き直すことができる。

## 3. ターン進行ステップ
スタンダードレギュレーションでは、アタックステップ後に「第2メインステップ」が行われる。

1.  **スタートステップ**: ターンの開始。
2.  **コアステップ**: ボイドからリザーブにコアを1個置く（先攻1ターン目は無し）。
3.  **ドローステップ**: デッキから1枚ドロー。
4.  **リフレッシュステップ**: 疲労カードの回復、トラッシュのコアをリザーブへ移動。
5.  **メインステップ**: 召喚、配置、使用、コア移動。
6.  **アタックステップ**: アタック（先攻1ターン目は無し）。
7.  **第2メインステップ**: 召喚、配置、使用、コア移動。アタックで疲労した後の盤面整理等（先攻1ターン目は無し）。
8.  **エンドステップ**: ターンの終了。

※**先攻1ターン目の制限**: コアステップ、アタックステップ、第2メインステップは行われない。
※**効果の発揮**: 第2メインステップ開始時にも「メインステップ開始時」の効果は発揮される。

## 3. 新ギミック・専門用語
*   **継召（けいしょう / Succession）**:
    *   トラッシュにある「EXシンボル」を持つカードを除外することで、その数だけ召喚・配置コストを軽減できる。
    *   トラッシュからのリソース活用による盤面復旧を支援する。
*   **ソウルマジック（Soul Magic）**:
    *   フィールドに指定された色のシンボルがある場合、本来のコストを支払う代わりに「ソウルコア1個」を支払うだけで使用できるマジック。
*   **真界放（しんかいほう）**:
    *   カウントに依存せず、指定されたコア数（またはソウルコアの有無）などでLvアップや追加効果を発揮する状態。

## 4. 使用可能カード（2026年5月時点）
*   **対象商品**: ブロックアイコン**「A」**および**「B」**以降のカード。
*   **主な商品リスト**:
    *   **ブースター**: 
        *   [26RBS01] 創世の鼓動、[26RBS02] 幻惑の翔風 (ブロックアイコン「A」)
        *   [BS76] エターナルブースター 永皇の輝き (2026年5月30日発売、契約編:証 第1章)
    *   **エントリーデッキ**: [26RSD01-06] 赫焔ノ風牙 〜 蒼淵ノ凶獣
    *   **コラボ**: [26RCB01] 仮面ライダー 運命の戦線、[26RSD07] 仮面ライダー AGENT OF DREAM

## 5. 契約編:証 (2026年度新ギミック)
*   **契約ネクサス (Contract Nexus)**:
    *   創界神ネクサスと同様の「神託」に加え、スピリットのように契約煌臨の対象や起点となることができる。
*   **神皇の復活**:
    *   『ダブルドライブ』10周年を記念し、十二神皇が「X」レアとして現代環境に合わせた効果でリメイクされている。
*   **期間**: 2026年4月1日 〜 2027年3月31日は「A」アイコンのみ使用可能。
*   **再録の扱い**: 旧アイコンのカードは、テキストが同じでも「A」アイコンが付与された再録版でない限りスタンダードでは使用不可。

## 5. デッキ構築のポイント
*   **EXシンボルの管理**: 継召を活用するため、EXシンボルを持つカードを適切にトラッシュへ送る手段が必要。
*   **ソウルコアの運用**: ソウルマジックの登場により、ソウルコアをリザーブに温存する重要性が高まっている。
*   **第2メインの活用**: アタック後に追撃の準備や防御のネクサス配置が可能なため、攻守のバランスが従来と異なる。

## 6. 効果解決のアルゴリズム
複数の効果が同時に誘発した場合や、解決中に新たな効果が発生した場合、以下のルールに従って処理する。

### 6.1 優先順位（基本）
1.  **常時発揮効果**: フィールドに存在する限り常に適用。誘発効果より優先。
2.  **誘発効果**: 「～したとき」等。**ターンプレイヤー**が自分・相手両方の全効果の解決順を決定する。

### 6.2 派生効果の「割り込み」
ある効果の解決によって新たに誘発した効果（派生効果）は、**待機中の他の効果よりも優先して解決**しなければならない。
*   派生効果の解決中にさらに別の効果が派生した場合、その新しい方を最優先する（連鎖）。
*   すべての派生効果が解消されるまで、元の解決順には戻れない。

### 6.3 テキストによるタイミングの差異
| 記述形式 | 解決ルール |
| :--- | :--- |
| **「Aし、Bする」** | AとBを同時に解決。 |
| **「A。その後B」** | A解決後、派生効果を待たずに直ちにBを解決。 |
| **「A。この効果発揮後B」** | Aから派生した効果（破壊時等）をすべて解決し終えてからBを解決。 |
| **「ただちに～」** | ターンプレイヤーの順序決定を無視し、最優先で割り込む。 |

### 6.4 不発と優先原則
*   **不発**: 解決時にカードがフィールドを離れている、またはLv条件を満たしていない場合は発揮されない。
*   **「できない」の優先**: 「～する」と「～できない」が衝突した場合、常に「できない」が優先される。
