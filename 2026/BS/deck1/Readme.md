ルール/フォーマット: バチルスピリット公式ルール/エターナル

要求: 
定義したデッキ(Deck#0,DEck#1..)において、1ターン目で「日下 チヒロ」1枚以上を手札に加える確率を算出。
各カードが手札に来た場合の効果を使用
組み合わせ数を算出
マリガンを行う選択肢も含めて。
計算手順を含めて記述せよ。
スプレッドシートとして計算式を生成
