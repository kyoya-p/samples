# 使用方法

## UIイメージ
![Address Book UI](./dialog.drawio.svg)

本アプリケーションは、上図のデザインに基づいた住所録管理画面を提供します。
- **一覧表示**: 登録された名前とメールアドレスを表形式で表示します。視認性を高めるため、行ごとに背景色を変えています。
- **削除**: 各行の右側にある "✕" ボタンをクリックすることで、その項目を削除できます。
- **追加**: 画面下部の入力フォーム（Name, Mail）に入力し、「Add」ボタンを押すことで新規登録できます。

## アプリケーションの実行
デスクトップアプリケーションを起動するには：

```shell
./amper.bat run jvm-app
```

## テストの実行
UIテストを含む自動テストを実行するには：

```shell
./amper.bat test
```

## プロジェクト構成
- **jvm-app**: デスクトップアプリケーションのエントリポイント。
- **shared**: メイン画面（Screen）やリポジトリなどの共通ロジックとUIコンポーネント。
