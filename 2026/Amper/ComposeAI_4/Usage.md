# 設定と実行

## 必要要件
- **JDK 21** (推奨)
  - ※ Java 25 環境ではネイティブライブラリのロードエラーが発生する場合があるため、Java 21 の使用を推奨します。

## Firestore 接続設定 (オプション)
本アプリケーションは Google Firestore との同期をサポートしています。
1. Firebase コンソールからサービスアカウントキー (`service-account.json`) をダウンロードします。
2. プロジェクトのルートディレクトリに配置します。
   - **配置あり**: Firestore モードで起動し、クラウド上のデータと同期します。
   - **配置なし**: インメモリモードで起動し、データはアプリ終了時にリセットされます。

## アプリケーションの起動
以下のコマンドでデスクトップ版アプリを起動します。
```shell
./gradlew.bat :jvm-app:run
```

## テストの実行
UI テストを含む全テストを実行するには以下を使用します。
```shell
./gradlew.bat test
```

# 機能概要

![Address Book UI](./dialog.drawio.svg)

本アプリケーションは、上図のデザインに基づいた住所録管理画面を提供します。
- **一覧表示**: 登録された名前とメールアドレスを表形式で表示します。視認性を高めるため、行ごとに背景色を変えています。
- **削除**: 各行の右側にある "✕" ボタンをクリックすることで、その項目を削除できます。
- **追加**: 画面下部の入力フォーム（Name, Mail）には、テスト用にランダムな値が自動的に入力されています。「Add」ボタンを押して登録すると、次のランダムな値が自動でセットされるため、手動入力の手間なく連続してテストデータを作成できます。
