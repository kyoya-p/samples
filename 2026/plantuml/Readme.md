# 実装手順

1. オリジナルレンダラサイト https://plantuml.com　を参照し、サンプルコードを採取しsamplesに格納。
2. 上記を参照し同等のsvgファイルを生成するindex.htmlコードを生成する
 - dead code, test codeは常に削除

# 機能
- [ ] PlantUML コード編集画面
  - [ ] テキスト編集した内容がGUIに反映される
  - [ ] GUI操作した内容がテキストに反映される

- [ ] オブジェクト
  - [ ] GUIで内容編集可能
  - ノード、リンク、がある。
- [ ] ノード
  - [ ] 形状:
    - [ ] 通常ノード
    - [ ] 開始ノード
    - [ ] 終了ノード

- [ ] リンク
  - [ ] ノードとノードを接続する
  - [ ] 矢印
    - [ ] 頭: なし、順方向、逆方向、双方向
    - [ ] 形状: -->, -->>, --|>, --x, --o,
  - [ ] スタイル: 実践, 破線,2重線
  - [ ] 線幅: Int
  - [ ] 接続元属性、接続先属性、 それぞれにラベルを付与

# 検証
samples以下のサンプルコードをそれぞれに付いて:
  originalレンダラでsvg画像を生成
  localレンダラでsvg画像を生成
  それぞれの画像を比較し、
    異なればlocalレンダラを修正し、画像生成から再試行
すべてのサンプルについてlocalレンダラを修正せずに施行完了したら検証終了
一つ以上のサンプルについてlocalレンダラを修正した場合、すべてのサンプルについて検証を再試行
