# スキャン送信画面
```
╭───────────────────────────────────────────────────────────────────────────────────────────────╮
│   Send                                                                                        │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│To:                                                                                            │
│  ╭───────────────────────────────────────╮                                                    │
│- │                                       │                [Enter]                             │
│  ╰───────────────────────────────────────╯                [Address Book]                      │                                                                  │
│- bob.681.smith@example.com                                [Remove]                            │
│- charlie.999.jones@example.com                            [Remove]                            │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                             [Send]  [Close]   │
╰───────────────────────────────────────────────────────────────────────────────────────────────╯
```

- [            ]: アドレス入力フィールド。
- [Enter]: (または'Enterキー') 確定、下のアドレスリストに追加。
- [Address Book]: アドレス帳選択画面を開き宛先を選択し下に追加。
- [Send]: 「~に送信しました」ダイアログを表示する。ダミー処理。
- [Close]: (または'ESC'キー) アプリ終了。

# アドレス帳選択ダイアログ
```
╭───────────────────────────────────────────────────────────────────────────────────────────────╮
│  Address Book                                                                                 │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│Name [      ]               Mail  [      ]                           Time v                    │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│Alice 101 Smith             alice.101@example.com                    2026-01-27 10:00          │
│Bob 681 Smith               bob.681.smith@example.com                2026-01-27 10:05          │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                       [Close] │
╰───────────────────────────────────────────────────────────────────────────────────────────────╯
-  Name, Mail, Time, のリストが位置合わせして表示される
- カラムタイトル
  - クリックしたカラムが選択される。選択されていないカラムはすべて非選択
  - 選択されたカラムは、ソート可能
  - 選択されたカラム(name,address)は、検索可能となり検索文字TextFieldが有効になる
  - ソートは昇順➜降順➜昇順とトグル
- アイテムをクリックしたらそのアイテムを選択し、送信画面状のリストに追加する。
- [Close]はダイアログを閉じる
