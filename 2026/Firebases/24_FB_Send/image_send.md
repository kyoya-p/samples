# スキャン送信画面
```
╭───────────────────────────────────────────────────────────────────────────────────────────────╮
│   Send Mail                                                                                   │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│    ╭───────────────────────────────────────╮                                                  │
│To: │                                       │              [Enter]  /  [Address Book]          │
│    ╰───────────────────────────────────────╯                                                  │
│- Alice 101 Smith - bob.681.smith@example.com              [Remove]                            │
│- Bob 681 Smith   - charlie.999.jones@example.com          [Remove]                            │
│                                                                                               │
│                                                                                               │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│ Send Log                                                                       [Clean Logs]   │
│ 2026/1/2 23:12                 xxxx@example.com, yyyy@example.com...                          │
│ 2026/1/2 20:12                 xxxx@example.com,                                              │
│                                                                                               │
│                                                                                               │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│  Status: Connected                                                        [<Send>]  [Close]   │
╰───────────────────────────────────────────────────────────────────────────────────────────────╯
```

- [            ]: アドレス入力フィールド。
- [Enter]: (または'Enterキー') 確定、下のアドレスリストに追加。
- [Address Book]: アドレス帳選択画面を開き宛先を選択し下に追加。
- [Send]: 「~に送信しました」ダイアログを表示する。ダミー処理。
- [Close]: (または'ESC'キー) アプリ終了。
- [Clean Logs]: サーバのSendLogをすべて消去する

# アドレス帳選択ダイアログ
```
╭───────────────────────────────────────────────────────────────────────────────────────────────╮
│  Select Address                                                                               │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│Name v [Alice    ]          Mail                                     Time                      │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│Alice 101 Smith             alice.101@example.com                    2026-01-27 10:00          │
│Bob 681 Smith               bob.681.smith@example.com                2026-01-27 10:05          │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
│                                                                                               │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                       [Close] │
╰───────────────────────────────────────────────────────────────────────────────────────────────╯
画面は"Name"が選択され、昇順ソート、検索文字列'Alice'が入力された状態。

-  Name, Mail, Time, のリストが位置合わせして表示される
- カラムタイトル
  - クリックしたカラムが選択される。選択されていないカラムはすべて非選択
  - 選択されたカラムは、ソート可能
  - 選択されたカラム(name,address)は、検索可能となりカラム内の検索文字TextField(1)が有効になる
  - ソートは昇順➜降順➜昇順とトグル
- アイテムをクリックしたらそのアイテムを選択し、送信画面状のリストに追加する。
- [Close]: (または'ESC'キー) ダイアログを閉じる
